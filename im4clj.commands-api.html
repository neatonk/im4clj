Full namespace name: im4clj.commands

Overview

Command function's. Uses im4clj.command/defcommands to define named series of named fn's which run the similarly named IM/GM commands.

Public Variables and Functions


animate

function
Usage: (animate options? files*)

Run the animate command with the given argument form. See IM/GM documentation for detailed usage.

Example Usage: 

(animate "cockatoo.*")
(animate "-map" "best" "cockatoo.*")
